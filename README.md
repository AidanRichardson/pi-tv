
# PI-TV

Self-hosted IPTV relay and web player for Raspberry Pi

## Deployment

To deploy this project run

```bash
  cd frontend
  npm run build
```
From project root directory:
```bash
  start.sh
```

## Development

To run the project in development mode run the following

```bash
  cd frontend
  npm run dev
```
From project root directory:
```bash
  dev.sh
```
